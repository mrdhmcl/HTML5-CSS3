Principais formatações

Negrito

Para o negrito, a melhor forma de destacar é utilizando a tag STRONG.

Itálico

Para o itálico, a tag EM.

Texto marcado

Para isso, podemos usar a tag mark. E inserir cor nela utilizando o style do CSS.

Palavras grande e pequena

Para a palavra aumentar de tamanho, é só utilizar a tag BIG.

Para reduzir, utilizar a tag small.

Palavras deletadas

Para criar o efeito: ninja excluído da vila da folha, utilizar a tag del.

Palavras inseridas

Para reinserir o ninja excluído, utilizar a tag ins.

Texto sobrescrito

(Tag "sup") Para inserir coisas do tipo 22 + 44 = x.

Texto subscrito

(Tag "sub") Para inserir coisas do tipo CO2.

Código fonte

Para isso, utilizar a tag code: paranoid android.

Além disso, é possível utilizar a tag pre para quebrar linhas de equações matemáticas. Por exemplo:

        50x + 50 = 1300
        50x = 1300 - 50
        50x = 1250
        5x = 125
        x = 125/5
        x = 25

Citações

Para inserir aspas curvas, utilizar a tag q. Como diria Mano Brown: o tempo ruim vai passar, é só uma fase, irmão.

Citações completas

Utilizar a tag blockquote.

Segundo Freud, em seu livro O mal-estar na civilização:

Se o desenvolvimento da civilização é tão semelhante ao do indivíduo, e se usa os mesmos meios, não teríamos o direito de diagnosticar que muitas civilizações, ou épocas culturais - talvez até a humanidade inteira - se tornaram neuróticas sob a influência do seu esforço de civilização?

Abreviações

Tag abbr: Um dia, a URSAL vai ser levada a sério e, neste dia, a América Latina deixará de ser a América Latrina.

Texto invertido

Tag bdo rtl: Eu queria acordar, mas dormi.
