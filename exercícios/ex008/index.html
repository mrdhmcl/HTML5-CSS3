Principais formatações

Negrito / Forte

Nesta frase, temos um termo em negrito usando a tag b (não semântica).

Nesta frase, temos um termo em destaque forte usando a tag strong (semântica).

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag i (não semântica).

Nesta frase, temos um termo em ênfase usando a tag EM (semântica).

Texto marcado

Além disso, podemos criar um texto marcado com a tag mark (semântica).

Agora, toda vez que utilizarmos a tag mark, o texto marcado aparecerá em vermelho crimson.

Palavras grande e pequena

Nesta parágrafo, iremos criar palavras grandes e pequenas.

Palavras deletadas

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado.

Palavras inseridas

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado posteriormente com a tag ins (semântica).

No entanto, também é possível utilizar a tag u para sublinhar e dar ênfase ao texto (método não semântico).

Texto sobrescrito

Para inserir coisas do tipo x20+3.

Texto subscrito

Para inserir coisas do tipo H2O.
